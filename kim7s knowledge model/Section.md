A section is a collection of knowledge about a broad subject. It can be:
* Theory
* Complex tool with many sub systems
* Language
* Framework

The section has a well defined structure:
* folders with subsection;
* 0Tip file with snippets regarding the broad subject
* 1Tags file with a the set of accepted tags for this broad subject
* 2Get Started
* 3Get Knowledge
* 4Get Help
* zSource file containing interesting links regarding the broad subject
