This category will show to the reader all information needed to get started with the subject. It will include a brief motivation, how to onboard to the technology and first steps.
It should have all information to get started with the technology and have it running with minimal effort. 
This category will not include in depth knowledge on how things works. 
It will include overview, tutorial on first tasks, and some guides on how to onboard.
This Category will always have 2 sections:
* Onboarding
* First Look
## Onboarding
This section should present all needed configurations to onboard the person/team to the subjecti. It should NOT contains any theory, only the steps to be introduced to the tool/technology/language/idea. It will have some pre-resquisites some times and it will have some suggestions on libraries or structures
## First Look
This section should navigate the user what to do first, how to start using the technology, or tool, or idea. Should NOT contains in depth theory, for the subject in hand.
